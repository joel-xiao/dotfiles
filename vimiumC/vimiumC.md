# Vimium C 操作指南

## 一、基础操作

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 显示帮助 | `?` | 打开 Vimium C 快捷键帮助页面 |
| 进入输入模式 | `Alt + /` | 在输入框、搜索框中启用输入模式 |
| 强制输入 `?` | `Alt + ?` | 在输入模式中输入 `?` 字符 |

---

## 二、标签页操作

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 创建新标签 | `Alt + t` | 在当前窗口创建新标签 |
| 重新加载标签 | `Alt + r` | 刷新当前标签 |
| 上一个标签页 | `c` | 切换到左边的标签 |
| 下一个标签页 | `v` | 切换到右边的标签 |
| 关闭当前标签 | `x` | 关闭标签并切换到附近标签 |
| 恢复最近关闭标签 | `z` | 恢复关闭的标签 |
| 关闭右侧标签 | `Xc` | 关闭当前标签右边的所有标签 |
| 关闭左侧标签 | `Xz` | 关闭当前标签左边的所有标签 |
| 关闭其他标签 | `Xa` | 关闭除当前标签之外的所有标签 |
| 固定/取消固定标签 | `Pp` | 切换标签固定状态 |
| 标签静音 | `Alt + m` | 切换标签静音 |
| 复制标签 | `Ctrl + s + d` | 复制标签 |
| 标签向右移动 | `>` | 将标签右移 |
| 标签向左移动 | `<` | 将标签左移 |
| 移动标签到新窗口 | `N` | 将标签移到新窗口 |
| 移动标签到无痕窗口 | `I` | 将标签移到无痕窗口 |
| 移动标签到下一个窗口 | `Alt + n` | 将标签移动到下一个窗口 |

---

## 三、滚动操作

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 向左滚动 | `h` | 页面向左滚动 |
| 向右滚动 | `l` | 页面向右滚动 |
| 向下翻页 | `j` | 页面向下滚动 |
| 向上翻页 | `k` | 页面向上滚动 |
| 滚动到顶部 | `gg` 或 `W` | 快速到页面顶部 |
| 滚动到底部 | `G` 或 `S` | 快速到页面底部 |
| 精确向上滚动 | `Ctrl + ↑` | 按像素滚动 |
| 精确向下滚动 | `Ctrl + ↓` | 按像素滚动 |
| 精确向左滚动 | `Ctrl + ←` | 按像素滚动 |
| 精确向右滚动 | `Ctrl + →` | 按像素滚动 |
| 翻页向上 | `w` | 向上翻页 |
| 翻页向下 | `s` | 向下翻页 |
| 向上滚动 | `q` | 页面向上滚动 |
| 向下滚动 | `e` | 页面向下滚动 |

---

## 四、导航操作

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 后退 | `A` | 页面后退 |
| 前进 | `D` | 页面前进 |
| 上一个历史标签 | `Q` | 切换到上一个标签 |
| 下一个历史标签 | `E` | 切换到下一个标签 |
| 切换到上一个标签（跨窗口） | `p` | 切换到上一个标签 |
| 切换到下一个框架 | `n` | 下一个框架 |
| 跳转到根目录 | `UU` | 当前页面根目录 |
| 向上跳转 | `Uu` | 当前路径上一级 |
| 父框架 | `Uf` | 跳转到父框架 |

## 五、Vomnibar

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 激活 Vomnibar | `o` | 在当前标签打开 Vomnibar |
| 在新标签打开 Vomnibar | `Alt + g` | 新标签打开 Vomnibar |
| 标签选择 Vomnibar | `t` | 在 Vomnibar 中选择标签 |
| 历史记录 | `H` | 在新标签打开浏览历史 |
| 书签 | `B` | 在新标签打开书签 |
| 输入 URL | `Fe` | 直接输入 URL 打开 |

## 六、LinkHints（链接操作）

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 激活链接提示 | `f` | 显示页面所有可点击链接 |
| 悬停激活链接 | `Ff` | 对悬停元素激活链接 |
| 下载链接 | `Fd` | 下载链接指向的内容 |
| 下载图片 | `FD` | 下载当前页面图片 |
| 打开图片 | `Fi` | 在新标签打开图片 |
| 复制图片 | `FC` | 复制图片到剪贴板 |

## 七、查找与可视模式

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 查找模式 | `/` | 进入查找模式 |
| 查找下一个 | `.` | 查找下一个匹配 |
| 查找上一个 | `,` | 查找上一个匹配 |
| 再次查找 | `Alt + .` | 查找上一次搜索结果 |

| 行可视模式 | `:` | 高亮整行文本 |

## 十一、Vimium C 内置命令

| 功能 | 快捷键/关键字 | 说明 |
|------|--------|------|
| Vimium 页面 | `v:` | vimium:// 页面操作 |
| Vimium 显示 | `v.d` | 显示 Vimium 页面 |
| Vimium 图片显示 | `v.i` | 显示图片 |
| 计算表达式 | `v.m` | 内置数学计算 |
| 求和 | `v.sum` | 计算求和 |
| 乘积 | `v.mul` | 计算乘积 |
| 平均值 | `v.avg` | 计算平均值 |
| 复制计算结果 | `v.mc` | 计算并复制 |
| 状态查询 | `v.st` | 查看标签状态 |
| 搜索 Vimium | `v.u` | Vimium 内搜索 |
| 复制 Vimium | `v.c` | 复制到剪贴板 |
| 解析搜索 | `v.p` | 重做搜索 |
| Bing 搜索 | `v.bi` | Bing 内搜索 |
| Lower Case | `:i` | 文本小写处理 |
